# Job Market Intelligence Dashboard

## Overview

The Job Market Intelligence Dashboard is an interactive data analytics project built using Python, Pandas, and Streamlit. It analyzes over 22,000 job postings to identify hiring trends, in-demand skills, top-paying roles, and leading hiring companies.

## Features

- Analyze 22,000+ job listings
- Top 10 Most In-Demand Job Titles
- Top 10 Most Requested Skills
- Top 10 Highest Paying Roles
- Top Hiring Companies
- Hiring Trends Visualization
- Interactive Dashboard using Streamlit

## Tech Stack

- Python
- Pandas
- Streamlit
- Matplotlib
- Excel Dataset

## Installation

Clone the repository:
git clone https://github.com/YOUR_USERNAME/Job-Market-Intelligence-Dashboard.git

Move into the project directory:
cd Job-Market-Intelligence-Dashboard

Install dependencies:
pip install pandas streamlit matplotlib openpyxl

Run the dashboard:
streamlit run dashboard.py

The application will open in your browser at:
http://localhost:8501

## Key Insights

- Data Analyst, Data Scientist, and Data Engineer are among the most demanded roles.
- Python, SQL, AWS, Excel, and Tableau are highly requested skills.
- Senior Data Science and Engineering positions offer the highest average salaries.
- Multiple organizations actively hire data professionals across various locations.
